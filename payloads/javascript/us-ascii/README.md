# US-ASCII
## What is US-ASCII?
US-ASCII is a type of text encoding used in the US. The only known server that uses US-ASCII is Apache Tomcat.
### What does <code>us-ascii.js</code> do>
<code>us-ascii.js</code> uses special characters to confuse XSS filters. An <code>XSS</code> <code>alert()</code> if vulnerable.
